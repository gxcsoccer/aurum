# Aurum 项目进展报告

> 最后更新：2026-03-27

## 一、项目目标

构建一个**自进化的量化投资系统**，借鉴 Karpathy 的 autoresearch 架构，让 LLM agent 自主迭代策略代码，最终目标是**跑赢 SPY 买入持有**。

## 二、架构

```
人类层（你）           →  scorer.py / program.md / config.yaml
Agent 层（qwen3.5-plus）→  strategies/strategy.py（唯一可修改文件）
基础设施层（不可变）     →  data.py / backtest.py / sandbox.py / llm.py
```

核心循环：LLM 提出变异 → 沙盒执行 → Walk-Forward 回测 → 评分 → 保留或丢弃。完全自主，无需人工干预。

## 三、迭代历程

### 阶段 1：单资产择时（SPY long/cash）

| 版本 | 做了什么 | 结果 | 问题 |
|---|---|---|---|
| v1 scorer + 基线动量 | 搭建整个系统，20 轮进化 | Sharpe 0.53→0.63, 回撤 16%→8.6% | 策略学会空仓躲避，年化才 5% |
| v2 scorer（加参与率惩罚） | 重新进化 30 轮 | Sharpe 1.09, 年化 10.7%, Holdout +10.9% | **跑不赢 SPY B&H (+18.9%)** |
| +50 轮深度进化 | 继续优化 | 最终 Sharpe 1.09, Holdout +10.9%, DD 4% | 到达局部最优，但本质上单资产择时在牛市无法赢 B&H |

**关键发现**：agent 自主发现了「双动量确认退出 + 成交量过滤 + 动量加速 + 最低持仓期」策略，逻辑完整且可解释。

### 阶段 2：均值回归策略 + 组合

| 做了什么 | 结果 |

**关键发现**：单资产择时的天花板很低——在牛市你每次空仓都在错过上涨。

## 五、Agent 自主发现的策略创新

从简单的 Dual Momentum 基线出发，经过约 50 轮自主进化，agent 发现了 4 项关键改进：

1. **多期动量加权**：1/3/6/12 个月加权（权重 1/2/4/6），比单一 12 个月回望更稳健
2. **波动率调整排名**：动量 / 波动率，优选风险调整后最佳资产
3. **相对 SPY 动量阈值**：进攻型资产需动量 >= SPY 动量才持有，确保只在有优势时出手
4. **防御型资产用纯动量排名**：避免 SHY 因波动率极低导致评分虚高

## 六、评分函数迭代教训

这是整个项目最大的收获——**每一版 scorer 都被 agent "破解"了**：

| 漏洞 | Agent 怎么利用的 | 修复方式 |
|---|---|---|
| 回撤惩罚太高 | 学会不交易（0 交易 = 0 回撤） | 加参与率惩罚 |
| 回撤优势加分 | 100% 持现金得满分 | 删除回撤优势加分 |
| 现金惩罚 >60% 才触发 | 保持 59% 现金刚好不触发 | 改为 >30% 渐进式惩罚 |

**Karpathy 的核心洞察完全适用**：你不是在编程策略代码，你是在编程 `program.md` 和 `scorer.py`——它们定义了 agent 的进化方向。

## 七、项目文件结构

```
aurum/
├── config.yaml                 # 7 资产池 + 时间窗口 + LLM 设置
├── program.md                  # 多资产轮动 agent 工作手册
├── loop.py                     # autoresearch 主循环
├── validate.py                 # holdout 验证（含预热）
├── portfolio.py                # 多策略组合器
├── .env                        # 百炼 API Key
├── infra/
│   ├── data.py                 # yfinance 多资产数据 + parquet/pickle 缓存
│   ├── backtest.py             # 多资产轮动回测 + walk-forward
│   ├── scorer.py               # v4 评分函数
│   ├── sandbox.py              # subprocess 沙盒
│   └── llm.py                  # 百炼 qwen3.5-plus 客户端
├── strategies/
│   ├── strategy.py             # 当前最优策略（多期动量加权轮动）
│   ├── strategy_momentum.py    # 保存的单资产动量策略
│   └── strategy_meanrev.py     # 保存的均值回归策略
└── experiments/
    └── results.tsv             # 完整实验日志
```
